# Project Portfolio Analytics Dashboard

A simple **Project Analytics Dashboard** built using **HTML, CSS, JavaScript, and Chart.js**.  
It visualizes insights from development and machine learning projects using interactive charts and summary metrics.

---

## Project Overview

This dashboard analyzes project data and presents insights such as:

- Total number of projects
- Total hours invested
- Average project difficulty
- Category distribution
- Hours spent per category

The goal is to transform raw project data into clear visual insights.

---

## Key Features

### Project Summary Cards
Displays quick analytics including:

- Total Projects Completed  
- Total Hours Invested  
- Average Project Difficulty  

---

### Category Distribution Chart
A **Pie Chart** showing how projects are distributed across categories such as:

- Frontend  
- Data Visualization  
- ML Regression  
- ML Classification  
- Deep Learning  

---

### Hours Investment Chart
A **Bar Chart** visualizing the number of hours spent on each project category.

---

## Technologies Used

- **HTML5** – Page structure  
- **CSS3** – Styling and layout  
- **JavaScript** – Data processing and DOM manipulation  
- **Chart.js** – Data visualization library  

---

## How the Dashboard Works

1. Project data is stored in a JavaScript array.
2. JavaScript calculates:
   - Total projects
   - Total hours invested
   - Average difficulty
3. Values are displayed in summary cards.
4. Chart.js renders:
   - Pie chart for category distribution
   - Bar chart for hours invested.
